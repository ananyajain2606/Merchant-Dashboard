## KPI Framework

### 1. Core Metrics
| KPI | Description | Sample Value |
|---|---|---|
| Total Transactions | Count of successful payment transactions | 84.2M (+9.3% MoM) |
| Gross Merchandise Value (GMV) | Total value of transactions processed (₹) | ₹241B (+7.8% MoM) |
| Active Merchants | Merchants with ≥1 transaction in the month | 14.6M (+5.1% MoM) |

### 2. Acquisition
| KPI | Description | Sample Value |
|---|---|---|
| New OB | Total new merchants registered in the period | 384K (+6.2% MoM) |
| New OB Active | New merchants who completed ≥1 transaction | 196K (51% activation) |
| DIY Channel | Self-onboarded merchants via app/web | 146K (+14.2% MoM) |
| SO Channel | Merchants onboarded via Sales Officers in the field | 238K (62% share) |

> **Activation rate benchmark:** 45–55%. Current: 51% ✅

### 3. Engagement
| KPI | Description | Sample Value |
|---|---|---|
| Super Engaged | Merchants transacting 20+ days in the month | 1.24M (+8.1% MoM) |
| Engaged | Merchants transacting 6–19 days in the month | 3.87M (+4.6% MoM) |
| Repeat | Merchants transacting for 2+ consecutive months | 6.12M (+3.2% MoM) |
| Revisits | Merchants returning after 1+ months of inactivity | 0.93M (+5.9% MoM) |
| Super Engaged 6+ Users | Super Engaged merchants with ≥6 unique payment users | 310K (+22% MoM) 🔥 |

> **Engagement funnel:** Revisit → Repeat → Engaged → Super Engaged → SE 6+

### 4. Retention
| KPI | Description | Sample Value |
|---|---|---|
| Churn (M-1 Txn) | Merchants active last month with 0 transactions this month | 1.12M (-2.8% MoM) ✅ |
| Winback | Previously churned merchants who transacted again | 0.44M (+12% MoM) 🔥 |

### 5. Device — Soundbox
| KPI | Description | Sample Value |
|---|---|---|
| SB Merchants | Merchants with an active Soundbox device | 5.69M (39% penetration) |
| Non-SB Merchants | Merchants transacting without Soundbox | 8.91M (61% headroom) |
| New SB Deployments | New Soundbox devices deployed in the period | 68.4K (+11% MoM) |

> **SB merchants show 2.4× higher retention** vs Non-SB — strongest device ROI signal.

### 6. Revenue
| KPI | Description | Sample Value |
|---|---|---|
| UPI CC Revenue | Revenue from UPI Credit Card transactions (MDR share) | ₹4.2Cr (+8.4% MoM) |
| Settle Now Active | Merchants using the instant settlement feature | 2.14M (+9.7% MoM) |
| Settle Now Revenue | Revenue earned from Settle Now fees | ₹6.8Cr (+11% MoM) 🔥 |
| Subscription Revenue | Revenue from device/platform subscription plans | ₹18.3Cr (+4.2% MoM) |

## Key Insights

| Domain | Insight |
|---|---|
| **Acquisition** | DIY channel growing 2.3× faster than SO — signals product-led growth opportunity |
| **Engagement** | Super Engaged 6+ users growing fastest at +22% MoM — highest LTV cohort to prioritise |
| **Retention** | Winback growing +12% MoM while Churn declining — re-engagement campaigns showing strong ROI |
| **Device** | SB penetration at 39% with 61% Non-SB headroom — major upsell opportunity |
| **Revenue** | Subscription is largest stream (62% share); Settle Now growing fastest at +11% MoM |
